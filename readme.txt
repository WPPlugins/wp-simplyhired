== Description ==

WP SimplyHired is a lightweight plugin that displays jobs from SimplyHired's partner solutions on your site.

Features:

* Lightweight plugin with minimal configuration needed
* Great for niche sites. You may specify a base query which ensures results contain only certain types of jobs (i.e. engineering or sales)
* Responsive

Requirements:

* SimplyHired API Key is required. This can be obtained [here](https://simply-partner.com/partners-signup)
* You must abide by the Simply Hired Partner [Terms Of Service](https://simply-partner.com/partner-terms/1)
* SimplyHired may serve a first-party tracking cookie to each user viewing the job site.

== Installation ==

1. Upload 'wp-simplyhired' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the settings section, include your API Key, agree to include SimplyHired's attribution as required by their Terms Of Service, and select the page for which to display jobs.
